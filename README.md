# 6-Bit CPU Design (Computer Architecture)

### Key Specifications:
1. **CPU Word Size**: 6 bits
2. **ALU Operations**: AND, ADD, ROR
3. **Number of Registers**: 5
4. **RAM Size**: 10 units
5. **Word Size (ISA & RAM)**: 16 bits
6. **Supported Instructions**: Register Mode, Immediate Mode, JMP, JL
